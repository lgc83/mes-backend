# mes-backend

MES(Manufacturing Execution System) 시스템을 위한  
Spring Boot 기반 백엔드 API 서버입니다.

## 기술 스택
- Java 17+
- Spring Boot
- Spring Security + JWT
- JPA (Hibernate)
- Gradle
- MariaDB / MySQL

## 주요 기능
- 회원 관리 (로그인 / JWT 인증)
- 생산 오더 관리
- 판매 오더 관리
- 공통 예외 처리
- 보안 설정 (Spring Security)

## 프로젝트 목적
- ERP / MES 백엔드 구조 이해
- 실무형 REST API 설계 경험
- 포트폴리오용 프로젝트
